Insert and format citations from your Mendeley reference library in LibreOffice Writer, similar to the Mendeley Cite add-in for Microsoft Word. Supports the Mendeley REST API (sign in with your own API app credentials) or a BibTeX file exported from Mendeley Reference Manager. Built-in styles: APA 7, Harvard (Cite Them Right), Chicago 17 author-date, IEEE, Vancouver. Citations are stored as reference marks with embedded metadata, so documents can be refreshed, renumbered and restyled at any time.
