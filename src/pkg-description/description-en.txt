Insert and format citations from your Mendeley reference library in LibreOffice Writer, similar to the Mendeley Cite add-in for Microsoft Word. Supports the Mendeley REST API (sign in with your own API app credentials) or a BibTeX file exported from Mendeley Reference Manager. Built-in styles: APA 7, Harvard (Cite Them Right), Chicago 17 author-date, IEEE, Vancouver. Citations are stored as bookmarks with their metadata embedded in document properties — a form that survives DOCX round-trips in both LibreOffice and Word — so documents can be refreshed, renumbered and restyled at any time.
